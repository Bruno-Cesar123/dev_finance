## 🎲 Rodando o Front End

```bash
# Clone este repositório
$ git clone <https://github.com/Bruno-Cesar123/dev_finance>

# Acesse a pasta do projeto no terminal/cmd
$ cd dev_finance
```

<p> Abra o arquino index.html no seu navegador para iniciar a aplicação</p>
